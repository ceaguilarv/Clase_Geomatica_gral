---

## Descripción

Este repositorio reúne los talleres, prácticas y ejercicios desarrollados en la asignatura **Geomática General**, enfocados en la gestión, transformación y análisis de datos espaciales.

El objetivo del curso fue comprender y aplicar flujos de trabajo profesionales en Sistemas de Información Geográfica (SIG), incluyendo interoperabilidad entre formatos, manejo de sistemas de referencia, procesos ETL espaciales y conexión con bases de datos geográficas.

---

## Software utilizado

- QGIS  
- PostgreSQL / PostGIS  
- Google Earth  
- Quarto  

---

## Estructura del repositorio

Cada carpeta corresponde a una entrega o taller específico:

- `Taller1_...` → Visualizacion SIG  
- `Taller2_...` → Geoportales y Servicios Web (OGC) 
- `Taller3_...` → Sistemas de Coordenadas 
- `Taller4_` → Conversión de formatos, ETL espacial y carga en PostGIS  

Cada taller incluye:

- Archivos fuente
- Productos generados
- Documento técnico en Quarto
- Render en HTML y/o PDF
- Evidencias gráficas

---

## Competencias desarrolladas

- Gestión de sistemas de coordenadas
- Conversión entre modelos ráster y vector
- Aplicación de filtros espaciales y alfanuméricos
- Procesos ETL espaciales
- Carga y consulta en bases de datos espaciales
- Interoperabilidad entre plataformas SIG
